# Contributing to Canvas Quarto Sync

Thank you for your interest in contributing! We welcome all types of contributions, from bug reports to new features. However, please note that this is a personal project and I may not be able to respond to all contributions in a timely manner.

## How to Contribute

1.  **Fork the Repository**: Create your own copy of the project.
2.  **Create a Branch**: `git checkout -b feature/AmazingFeature`.
3.  **Commit Changes**: `git commit -m 'Add some AmazingFeature'`.
4.  **Push to Branch**: `git push origin feature/AmazingFeature`.
5.  **Open a Pull Request**: Describe your changes and why they are needed.

## Reporting Bugs

Please include:
*   Your OS and Python version.
*   The exact error message/traceback.
*   Steps to reproduce the issue.
*   If you have an idea of how to fix the issue, please include that as well.

## Style Guide

*   Follow PEP 8 for Python code.
*   Keep handler logic modular (add new handlers in `handlers/`).
*   Ensure all new features are documented in the User Guide.
